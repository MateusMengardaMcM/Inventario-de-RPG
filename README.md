# Inventário de RPG

Projeto voltado para fins educacionais, com intenção de aprender javascript, focando nos conceitos das arrays e matrizes.

## Aprendizados

Compreendi os conceitos das arrays e das matrizes.

## Funcionalidades

-digitar o nome de um dos seguintes itens para adicioná-los ao seu inventario:
espada, anel, poção, livro, escudo.
-caso digite errado a página exibira uma mensagem de erro.

## Stacks utilizada

**Front-end:** Html, css & JavaScript.



## Deploy

Para fazer o deploy desse projeto faça um clone do repositório e execute o arquivo em index.html.

```bash
  git clone https://github.com/MateusMengardaMcM/Inventario-de-RPG
```
